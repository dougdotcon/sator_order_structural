## Program Architecture

The Sator research established a foundational result:

$$\text{geometric symmetry} \Rightarrow H(M \mid G) \ll H(M) \Rightarrow \rho(t) > 0 \text{ under noise}$$

The question is now: **is this a property of the Sator, or a universal structural law?**

Each research line below is an independent escalation path from combinatorics toward testable physical principles.

## What Each Line Needs to Become a Paper
